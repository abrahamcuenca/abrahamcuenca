## Projects

### [Wayfarer Framework](https://github.com/PixelPilgrimStudios/wayfarer-framework)
Open-source Godot 4.x framework for rapid game development. Modular scene loading, cross-platform input (keyboard, mouse, gamepad, touch), settings and localization system with 7-language support, in-editor translation tools, and a developer console.

### [The Drug Name Emoji Game](https://uhhhehheh.com/2025/05/06/the-drug-name-emoji-game)
Web-based educational rebus game for pharmacy students built in ~40 hours of weekend work in collaboration with Dr. Timothy Gauthier of idstewardship.com. Custom shaders, in-game store, lives/timer system, 40 drug name puzzles. Built with Godot 4 and the Wayfarer Framework.

### [Beli's List](https://belislist.com)
Cross-platform mobile app for planning quinceañera events — wish lists, item tracking, and event organization. Co-created with a collaborator.
`TypeScript` `React Native` `GraphQL` `Apollo` `PostgreSQL` `Firebase` `Appwrite` `Tailwind CSS`

### CarCode at Edmunds.com
Consumer-facing vehicle marketplace frontend. Built and maintained features across the product.
`React` `TypeScript` `AngularJS` `JavaScript` `Bootstrap`
